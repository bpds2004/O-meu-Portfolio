## Como correr (adicionado pelo dono do portfólio)

Este projecto é uma aplicação React criada com Vite. Se abrires `index.html` diretamente e vês uma página em branco, executa os seguintes passos na pasta do projecto:

```bash
npm install
npm run dev
```

Isto iniciará o servidor de desenvolvimento do Vite e permitir-te-á abrir o URL mostrado (ex.: `http://localhost:5173`).

Alternativa para produção:

```bash
npm run build
npm run preview
```

Se preferires que seja gerada uma versão estática para colocar no site principal, diz-me que eu faço o build e copio a pasta `dist`.
