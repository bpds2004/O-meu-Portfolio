Este projecto é uma app React + Vite

Se esta página aparecer em branco ao abrir localmente, significa que as dependências não foram instaladas e o servidor de desenvolvimento do Vite não está em execução.

Passos rápidos para correr localmente (na pasta deste projecto):

  1. Abra o terminal na pasta bot-academico-utad-main/bot-academico-utad-main
  2. Execute npm install
  3. Execute npm run dev e abra o URL mostrado (ex.: http://localhost:5173)

Alternativas: npm run build + npm run preview para servir os ficheiros de produção.
